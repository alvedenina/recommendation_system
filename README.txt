Реализован проект рекомендательной системы постов для социальной сети на основе тестового датасета.
1) Обработка данных. Даны 3 дата фрейма:
    - таблица пользователей
    - таблица постов (к базовой таблице добавлены новые признаки. Векторизированы тексты постов с помощью
     модели-транформера  DistilBERT. Векторы постов разбиты на кластеры с помощью алгоритма kNN и получены
     веса для каждого поста для каждого кластера как дополнительный признак. Данный этап можно посмотреть в файле
     df-final-project-kaggle.ipynb)
    - таблица действий пользователей
Таблицы обработаны и объединены в одну. (data_processing.py)

2) Для взаимодействия с системой написан сервер на FastApi с помощью SQLAlchemy. База данных - PostgreSql.
(папка server)
Обучена модель рекомендаций на основе алгоритма градиентного бустинга CatBoost. (learning_model.py)

3) Функция рекомендаций реализована в модуле app.py.
