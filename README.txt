В ходе обучения использовала библиотеки: pandas, numpy, matplotlib, seaborn, scipy, sklearn, PyTorch и SQLAlchemy.
Реализован проект рекомендательной системы постов для социальной сети на основе тестового датасета.
1) Для взаимодействия с системой написан сервер на FastApi с помощью SQLAlchemy. База данных - PostgreSql.
Обучена модель рекомендаций на основе алгоритма градиентного бустинга CatBoost.


•	Для улучшения предыдущей модели добавлены новые признаки. Векторизированы тексты постов с помощью модели-транформера DistilBERT. Векторы постов разбиты на кластеры с помощью алгоритма kNN и получены веса для каждого поста для каждого кластера как дополнительный признак.


•	На основе дополнительных данных (лайки пользователей после рекомендаций) проведен сравнительный анализ двух вариантов и А/В тестирование на предмет значимости изменений.
